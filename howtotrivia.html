How to Create a Quiz

I have decided to create a trivia quiz because I thought that the skill of making a quiz in HTML and javascript would be helpful later in my life. For example I could make a signup form on a social media website that uses similar concepts. To create this quiz, I came up with 21 random questions and programmed it on javascript to check whether the answers were correct and come up with a score for the user. I am going to describe how I made the quiz and why I made the quiz this way.

Objective 1: How to Determine Correct Answer

Step 1: Create a Submit Button

          <input id="Submit" class="answers" type="button" value="Submit Answers">

To create a Submit button, give the button an id so that you can use it in the Javascript. The type has to be "button" and the value attribute will be the text shown on the button. This doesn't make the button work yet, we have to do that in Javascript.

Step 2: Get references to the elements that you created in the HTML

          var btn = document.getElementById("Submit");
          var scoring = document.getElementById("score");

Use getElementById to manipulate HTML elements in Javascript. getElementById searches for an element with the specified ID, and returns a reference to it.

Step 3: Make Submit button work and store all the radio buttons that are checked

        btn.addEventListener("click", function() {
            var radios = document.querySelectorAll('input[type="radio"]:checked');

To make the submit button work, add an event listener that fires when the button is clicked. In the event listener, add a variable that stores references to the clicked radio buttons in an array.

We created a for loop because this loop checks what radios.length is, which in this case is 21 and runs the code under it for all the questions. score++ adds 1 to the score every time we get a question correct.

Objective 2: How to Display the Score

Step 1: Link to Javascript in HTML

        <script type="text/javascript" src="trivia.js"></script>

To link to Javascript in HTML, the type has to be “text/javascript” and the src is the name of your Javascript file. This will execute that file's code on this page.

Step 2: Create div to display the score

        <div style="background-color: orange" class="trivia" id="score"></div>

The div element is a generic "division" element with few default styles. To create this div, give the div an id so that you can use it in the javascript and then give the div a background color. Later, we'll use Javascript to put the score in this div.

Step 3: Make score show on the bottom

        scoring.innerHTML = score + " /21 ";

We have to use the innerHTML property because we created a div to show the score in the HTML file. We created a score variable before that checks how many questions we got correct. Put that number over 21 for the final score.

Objective 3: How to show correct answer after Submit button is clicked

Step 1: Link to CSS in HTML

         <link rel="stylesheet" href="styles.css" type="text/css"/>

To link to CSS, the type has to be “text/css” and the href or src has to be the name of your CSS file.

Step 2: Put all of the correct answers into a span tag with class "correct" 

        <input type="radio" name="answer1" class="correct" value="1"><span class="correct">Nothing</span>

We need to create a span tag so that the text itself will be its own element. Give the span tag a class because we need to style it in CSS.

Step 3: Add the class quiz-complete to parents of the radio-buttons

        radios[i].parentNode.className += " quiz-complete";

Concatinate the class to the end of the className property. Be sure to add a space before the class Do this to apply the styles to the correct answer.

Step 4: Use CSS to highlight correct answer after submit button is clicked

      .quiz-complete .correct{
        background-color: lightgreen;
      }

In the CSS we have to highlight the correct answer green after the submit button is clicked. We have to do this so that the user will know what the correct answer is if they got the answer wrong.
